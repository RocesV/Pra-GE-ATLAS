Summary

This tool is designed to further explore, in an interactive way, the WGCNA co-expression networks computed in the Pra-GE-ATLAS manuscript for our reference species, Pinus radiata. All tissue/stress conditions were included in each network. For modules (M) with a large number of edges, a more stringent threshold was chosen. Please refer to the description of each module for clarification. Extended information about the modules is provided in the manuscript. The Gene Expression - Level Network is displayed by default. Full network is only available in the Gene - Expression Level Network due to the high connectivity of the Protein Abundance - Level Network. Users can switch between displays by clicking on the following options:

Gene Expression - Level Network     Protein Abundance - Level Network

*Please be aware that rendering large networks corresponding to the full network or the biggest modules could take some time (< 1 min).
